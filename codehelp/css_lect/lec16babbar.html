inline vs division

Inline: Elements with display: inline; do not start on a new line. They flow inline with the content. They only take up as much width as necessary and do not force a line break after the element. Horizontal space to the left and right of the element is respected, and the next element can appear on the same line.
example:-span a img
Block: Elements with display: block; start on a new line and take up the full width available. They create a new block-level box and force a line break after the element. The width, height, padding, and margin properties can be explicitly set. example : div all heading h1 to h6 p form header footer section
